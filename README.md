# SQL Project – Online Retail Sales Analysis

## 📌 Problem Statement

Retail businesses generate a large amount of sales data, but often struggle to get useful insights from it. This project focuses on designing a database and using SQL queries to analyze sales performance in a simple and structured way.

---

## 🎯 Objectives

* create a relational database for an online store
* store customer, product, and order details
* use sql queries to extract meaningful insights

---

## 🗂️ Database Tables

### customers

* customer_id
* name
* city

### products

* product_id
* name
* category
* price

### orders

* order_id
* customer_id
* date

### order_items

* order_id
* product_id
* quantity

---

## 🔧 key tasks performed

* find top-selling products
* identify most valuable customers
* calculate monthly revenue
* perform category-wise sales analysis
* detect inactive customers

---

## 📊 expected outcome

* a well-structured database
* simple and clear sql queries
* useful insights like:

  * best-selling products
  * high-value customers
  * monthly sales trends
  * category performance
  * inactive users

---

## ✅ conclusion

this project helps in understanding how sql can be used to analyze real-world business data. it also improves skills in joins, grouping, and basic data analysis.
